VK Broadcast POC

Get Community Token

1. Go to your VK Communities

2. Select your community → Manage → API usage

3. Create token with wall permissions

Long-lived token for posting to community walls
Negative number for groups (e.g., -123456789)

Get User Token

1. Create app at VK Developers

2. Choose "Standalone" type

3. Get your App ID and use it below

Get from vk.com/dev
Protected key from app settings
Can be obtained via OAuth or Implicit Flow
Positive number for user pages

Generate Token via Implicit Flow

Enter your App ID and select permissions to generate authorization URL

Default: https://oauth.vk.com/blank.html
After Authorization
After authorization, copy access_token from the redirect URL
Negative for groups, positive for user pages
VK API version (recommended: 5.131 or higher)
